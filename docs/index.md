# GenAIOps独習ロードマップ 〜GeminiとLangChainで始めるAIアプリ開発〜

本書は、何らかのプログラミング経験があるエンジニアが、Pythonを使ったAIアプリケーション開発とその運用（GenAIOps）を1ヶ月で体系的に学ぶためのロードマップです。

## この本のコンセプト
- **Learn by Building**: 他言語の経験を活かしながら、Pythonのエコシステムに飛び込み、実践的にAIアプリを構築します。
- **Gemini First**: コストパフォーマンスと性能に優れたGoogleのGemini 2.5 Flashをメインに使用します。
- **Ops Mindset**: 単にアプリを動かすだけでなく、「観測性」「評価」「改善」のサイクルを回すGenAIOpsの考え方を実践します。
