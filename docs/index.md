# TypeScriptエンジニアが挑む！GenAIOps独習ロードマップ

本書は、普段TypeScriptやNode.jsを使って開発しているWebエンジニアが、Pythonを使ったAIアプリケーション開発とその運用（GenAIOps）を1ヶ月で体系的に学ぶためのロードマップです。

## この本のコンセプト
- **From TypeScript to Python**: 常にTypeScript/Node.jsの概念と比較しながら解説します。
- **Gemini First**: コストパフォーマンスと性能に優れたGoogleのGemini 2.5 Flashをメインに使用します。
- **Ops Mindset**: 単にアプリを動かすだけでなく、「観測性」「評価」「改善」のサイクルを回すGenAIOpsの考え方を実践します。
